## 🐾 API 新增中国线路（由 犽可 提供）🎉

为提升国内用户的访问体验，我们新增了一条由 **犽可** 提供的中国加速线路！  
相较于主站，这条线路在中国大陆地区拥有更快的响应速度和更稳定的访问表现～✨

---

### 📌 接口地址

- 🌍 **主站（国际线路）**：  
  `https://api.furry.ist`

- 🇨🇳 **中国线路**（推荐中国用户使用）：  
  `https://furry.axzt.top`

📌 *当前中国线路暂不支持自动横竖屏适配，需自行处理布局～*

---

### 🔄 随机图片获取方式

添加参数 `?format=json` 可获取随机图片的 JSON 格式链接，方便程序调用！

**示例：**
```
https://furry.axzt.top?format=json
```
